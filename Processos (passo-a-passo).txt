Processos:

1) Base de dados: scripts que obtenham os dados da fonte automaticamente e de forma periódica, gerando xlsx na pasta correspondente do GitHub do projeto (T25, ou );
2) Geração de Link do Excel (xlsx) para ser atribuido a botão de download da figura correspondente na página: clicar sobre identificação alfanumérica da figura no GitHub do projeto e, no canto superior direito, clicar com o botão da direita sobre o botão "raw", para selecionar "copiar link". Copiar o link no lugar correspondete do arquivo "Checklist...xlsx".
3) Geração da figura no Power Bi (pbix): entrar na conta pessoal do Power Bi no Office da UFS e inserir o link xlsx dos dados a partir dos quais serão gerados a figura em "Obter Dados", "Web". Elaborar e figura e salvá-la no Meu workspace.
4) Geração do link da figura do Power Bi, para ser atribuida à figura na página: Com arquivo aberto no Power Bi da conta pessoal no Office da UFS, clicar em "Arquivo"-"Inserir Relatório"-"Publicar na web". Clicar no botão verde de cima (Link para ser enviado por e-mail). Copiar o link no lugar correspondete do arquivo "Checklist...xlsx". Observação: todos os visuais serão disponibilizados e as figuras serão individualizadas para publicação pelo pessoal de Ciência da Informação.
5) Geração de relatório textual de cada figura: Abrir o link do Power Bi e fazer um arquivo txt com os comentários descritivos de cada figura. Salvar o arquivo na pasta correspondente do GitHUb do projeto. Clicar sobre identifcação alfanumérica da figura no GitHub do projeto e, no canto superior direito, clicar com o botão da direita sobre o botão "raw", para selecionar "copiar link". Copiar o link no lugar correspondete do arquivo "Checklist...xlsx".
6) Geração do link do comentário txt de cada figura: clicar no link do arquivo txt do GitHub do projeto e, no canto superior direito, clicar com o botão da direita sobre o botão "raw", para selecionar "copiar link". Copiar o link no lugar correspondete do arquivo "Checklist...xlsx".
